# 🌬️ «Ваш Консул» — Telegram-бот по климатическому оборудованию Daichi

**«Ваш Консул»** — это интеллектуальный Telegram-бот, который выступает в роли технико-коммерческого консультанта по климатическому оборудованию.  
Бот специализируется на продукции бренда **Daichi** (бытовые, полупромышленные и промышленные системы), но также обладает общими знаниями по всем существующим брендам.

Бот использует современные LLM (DeepSeek или YandexGPT) для генерации осмысленных ответов, поддерживает контекст диалога, может подбирать оборудование по площади, типу помещения, бюджету и другим параметрам. Работает 24/7 на serverless-инфраструктуре Yandex Cloud.

## 🎯 Возможности бота

- ✅ **Подбор оборудования** – по площади, назначению (дом/офис/серверная/промышленность), бюджету.
- ✅ **Технические консультации** – характеристики линеек Daichi (Peak, Carbon, City Line, напольно-потолочные, канальные).
- ✅ **Сравнение моделей** – анализ двух или более моделей.
- ✅ **Коммерческие рекомендации** – учёт цены, энергоэффективности, сезонности.
- ✅ **Работа с контекстом** – бот помнит последние 10 сообщений диалога.
- ✅ **Выбор AI‑провайдера** – можно переключаться между **DeepSeek** (по умолчанию) и **YandexGPT**.
- ✅ **Команды Telegram** – `/start`, `/help`, `/clear`, `/daichi`, `/select_ai`.
- ✅ **Поддержка inline-кнопок** – удобное переключение моделей.
